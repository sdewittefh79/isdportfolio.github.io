Tools Used for the Build

Coursew were authored in Articulate Storyline 360 or Articulate Rise. All videos, icons, and illustrations are from the Articulate Content Library. Stock images were downloaded from Articulate Content Library or Pixabay. Adobe Photoshop was used to add a radial blur effect to the spinning game wheel button. The background music was downloaded as an MP3 file from eLearning Brothers. Audacity was used to create and edit the audio narration. It was also used to create the game wheel spin clicking sound. Adobe Illustrator was used to create the Happy Hour Logo in the Three Drivers of Team Success Course
